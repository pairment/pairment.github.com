--- layout: docs title: Link text should not begin with redundant text ---
Since screen reader users already hear links labeled as "link," starting a link text with a similar word causes redundancy. Try to change links to more natural language by removing repetitive words. We currently consider the following words repetitive:
  • link
  • go to
  • click here
  • more

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 2.4.9
and WCAG techniques:
  • F84
