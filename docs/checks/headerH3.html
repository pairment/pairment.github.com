--- layout: docs title: Headings following a h3 should be a h1, h2, or h4 ---
Headings on the page should be ordered sequentially. For example, a heading level 1 (h1) should not be immediately followed by an h5. The following is an example of a good heading order:
  • h1
  • h2
  • h3
  • h3
  • h2
  • h3
  • h4
  • h5

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 2.4.6
and WCAG techniques:
  • G130
