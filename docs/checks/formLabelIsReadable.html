--- layout: docs title: Form labels must be readable ---
Any labels used to describe elements of a form (i.e. a text area or a checkbox) must be readable. Empty labels are not helpful for users with screen readers.  

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 1.1.1
  • 1.3.1
  • 2.1.1
  • 2.1.3
  • 3.3.2
  • 4.1.2
and WCAG techniques:
  • H44
  • F68
  • H91
