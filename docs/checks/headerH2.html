--- layout: docs title: Headers following a h2 shoud be a h1 or h3 ---
Headers on the page should be ordered sequentially. For example, a header level 1 (h1) should not be immediately followed by an h5. The following is an example of a good header order:
  • h1
  • h2
  • h3
  • h3
  • h2
  • h3
  • h4
  • h5

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 2.4.6
and WCAG techniques:
  • G130
