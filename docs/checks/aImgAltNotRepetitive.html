--- layout: docs title: Don't repeat link text in an alternative text ---
When including an image within a link, the alternative text is read within the link. This can cause a link to be read twice. For example:   
<a href="/contact">Contact us <img src="mailbox.png" alt="Contact us"/></a>
Would read as "Contact us Contact us."  If an image is repetitive or decorative, set the alternative text to an empty value.

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 1.1.1
  • 2.4.4
  • 2.4.9
and WCAG techniques:
  • H30
