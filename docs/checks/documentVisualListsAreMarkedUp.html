--- layout: docs title: Define lists appropriately ---

Use an ordered or unordered list to define a logical group of items. Placing a bullet or other character at the start of a new line, instead of using a list, makes it more difficult to scan the group of items. For example, instead of:  

<p>
  • Item one<br/>
  • Item two<br/>
  • Item three
</p>

Use:  

<ul>
  <li>Item one</li>
  <li>Item two</li>
  <li>Item three</li>
</ul>

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 1.3.1
and WCAG techniques:
  • H28 
  • H48
  • T2
