--- layout: docs title: Headings should contain text --- Users with screen readers use headings like h1 to navigate the structure of a page. All headings should contain either text, or images with appropriate alternate text.  

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 1.3.1
  • 2.4.10
and WCAG techniques:
  • G141
