--- layout: docs title: Give definitions for all abbreviations ---

For every abbreviation on your page, you should provide a clear definition. Use the abbr or acronym tag. For example:  

<p>
  The <abbr title="World Wide Web Consortium">W3C</abbr> defined accessibility guidelines.
</p>

If you reuse an acronym on the same page, you only need to define it the first time it is used.

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 3.1.4
and WCAG techniques:
  • H28
