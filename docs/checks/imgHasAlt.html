--- layout: docs title: Images must have alternative text ---

Users with screen readers or a visual impairment use the alternative text of an image to understand its meaning. Alternative text should be short, descriptive, and convey the content of the image itself. For example, if there is an image of a pumpkin, the alternative text should not be "Cat." For example:

<img src="pumpkin.png"/>

Should instead be:  

<img src="pumpkin.png" alt="A pumpkin"/>

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 1.1.1
and WCAG techniques:
  • F65
  • H37
