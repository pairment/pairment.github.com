--- layout: docs title: Links should be readable ---

When a link does not contain any readable text, any images should include a readable alternative text. If no elements within the link are readable, then the link will be read as "link" without any context as to where it goes.

<a href="/contact"><img src="mailbox.png" alt=""/></a>

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 1.1.1
  • 2.4.4
  • 2.4.9
  • 4.1.2
and WCAG techniques:
  • H30
  • H91
  • H2
  • F89
