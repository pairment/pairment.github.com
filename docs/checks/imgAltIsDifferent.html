--- layout: docs title: Alternative text should not be the filename ---
The alternative text of an image should not just be the filename. Alternative text is to describe an image, and the filename usually includes confusing elements such as ".png." For example, change: 
<img src="pumpkin.png" alt="pumpkin.png"/>
to:   
<img src="pumpkin.png" alt="A pumpkin"/>

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 1.1.1
and WCAG techniques:
  • H37
