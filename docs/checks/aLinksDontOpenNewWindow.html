--- layout: docs title: Links do not open a new window without warning ---

Opening new windows when the user does not expect them can cause confusion and disrupt navigation. 

For example:  

<a href="/contact" target="_blank">Contact us</a>

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 3.2.5
and WCAG techniques:
  • F22
