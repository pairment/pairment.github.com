--- layout: docs title: Link text should be useful ---
Because many users of screen-readers use links to navigate the page, providing links with text that simply read "click here" gives no hint of the function of the link. We consider links that only contain the following text as not useful:
  • click here
  • click
  • more
  • here
  • read more
  • download
  • add
  • delete
  • order
  • view
  • read
  • image

Guideline information:

This check applies to WCAG 2.0 Sections:
  • 1.1.1
  • 2.4.4
  • 2.4.9
and WCAG techniques:
  • H30
