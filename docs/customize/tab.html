--- layout: docs title: Changing the style of the accessibility tab ---

The accessibility tab has several utility classes to allow you to change it. By default, the style of the accessibility tab is:

._pairment-tab {
	position: fixed;
	top: auto;
	bottom: 0;
	right: 20px;
	border-radius: 8px 8px 0 0;
	background-color: #004FAD;
}

._pairment-tab button {
	background: transparent !important;
	border: none !important;
	color: #fff;
	font-size: 15px;
	padding: 5px 15px;
	font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "Roboto", "Oxygen", "Ubuntu", "Cantarell", "Fira Sans", "Droid Sans", "Helvetica Neue", sans-serif !important;
	cursor: pointer;
}

However, the tab has an extra, unused class called "_pairment-tab-override" that exists so you can add a more specific selector. For example:

._pairment-tab._pairment-tab-override {
  background: #f00;
}
