--- layout: docs title: Javascript --- Pairment works by adding Javascript to your page. If you aren't using our Wordpress or Drupal plugins, then you will have to do a little coding to control when and for whom Pairment's accessibility feedback is displayed.

Settings

You can change the default behavior of pairment by setting a pairmentOptions object.

These settings will override any settings made in your Pairment site:

var pairmentOptions = {
  tab: true,
  tabLanguage: {
    show : "Show feedback",
    hide : "Hide feedback"
  }
};

Available settings

  • disable - Set to true to completely disable Pairment.
  • tab - Set to true to show the accessibility feedback tab, false to hide it.
  • autoRun - Set to true to run accessibility checks immediately.
  • selector - A CSS selector of the content area to check.
  • tabLanguage - Change the language of the accessibility tab. This should be an object with two properties:
    • show - A string used to show accessibility feedback
    • hide - A string used to hide accessibility feedback

Events

If you want to customize when a user sees accessibility feedback, you can disable the accessibility tab and trigger Pairment to run using events pairment.run and pairment.hide. These events are bound to the window object, and can be triggered like this:

window.dispatchEvent(new Event('pairment.show'));

Or in jQuery:

$(window).trigger('pairment.show');
